[about]
about=QGIS MCP HTTP starts a localhost-only Streamable HTTP MCP endpoint directly inside QGIS. It copies the existing QGIS MCP command handlers and exposes them at /mcp so clients can inspect projects, manage layers, run processing algorithms, render maps, and optionally execute PyQGIS code without a separate local stdio server checkout.
